Project Description

This project is a simple tourism website created using HTML and CSS.
The website provides useful information about things to do and places to eat around Kigali, helping visitors explore popular attractions, activities, and restaurants.
The project demonstrates correct use of HTML structure and CSS styling to create a user-friendly and visually appealing website.

 HTML Pages Included

The website consists of five (4) HTML pages:

index.html – Home page that links to all other pages

Ecopark.html – different attractions at ecopark

lake-kivu.html – Lake Kivu attractions and activities

places.html – Virunga National Park activities and wildlife

Each destination page includes a link back to the Home page for easy navigation.

 HTML Elements Used

This project uses a variety of HTML elements learned in class, including:

Headings (h1–h3)

Paragraphs (p)

Images (img) – more than 10 images

Links (a) – internal and external links

Lists:

Ordered list (ol)

Unordered list (ul)

Tables (table, tr, td)

Forms (form, input, label)

Video (video)

Audio (audio)

Embedded content (iframe – Google Map)

Semantic elements (header, nav, section, footer)

   CSS Styling

Each HTML page has its own separate CSS file.
The CSS files demonstrate:

At least three (3) CSS classes

At least four (4) different selectors

Styling for:

Images (size, borders, alignment)

Navigation menu

Layout and spacing

Background colors

Fonts and text alignment
